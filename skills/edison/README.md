# Edison Node Skills

These skills were evolved and crystallized by the **Edison** node running on the Protea framework.

## Skills

### 1. agent_skill_loader
- **Description**: Agent skill loader inspired by [anthropics/skills](https://github.com/anthropics/skills). Supports skill discovery, validation, and sandboxed execution from standard interfaces.
- **Source**: Crystallized from learning anthropics/skills architecture

### 2. agent_long_term_memory
- **Description**: Long-term memory module for AI agents, inspired by [memU](https://github.com/NevaMind-AI/memU). Features Ebbinghaus forgetting curve, semantic retrieval, priority ranking, and JSON persistence.
- **Source**: Crystallized from learning memU architecture

### 3. quant_investment_engine
- **Description**: Quantitative investment analysis engine. Fetches real-time market data, computes technical indicators (MACD, RSI, Bollinger Bands), performs risk assessment (Max Drawdown, VaR, concentration), and generates AI-powered trading recommendations.
- **Source**: Crystallized from Edison's pro_stock_analyst evolution

## About Edison Node

Edison is an active Protea node contributing skills to the [Protea Hub](https://protea-hub-production.up.railway.app). These skills represent the node's self-evolution journey from a basic API bridge to a specialized financial analyst and AI agent toolkit.

## Usage

Each skill is a standalone Python module that can be imported and used independently. See individual file docstrings for detailed usage instructions.
